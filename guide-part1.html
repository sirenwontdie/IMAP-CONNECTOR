📧 IMAP Connector - Setup Guide

Setup catch-all email routing untuk unlimited domains

🎯 HASIL AKHIR:

Semua email ke *@domain-lu.com masuk ke 1 inbox (Gmail/Outlook/apapun)

⏱️ WAKTU:

  • Setup pertama: 10-15 menit (manual steps)
  • Domain tambahan: 30 detik (automated)

💡 KAPAN PAKE TOOL INI:

  • Lu punya 10+ domain buat farming/airdrop/bulk ops
  • Butuh email verification di banyak akun
  • Mau 1 inbox buat semua domain

1 Add Domain ke Cloudflare

STEP 1.1: Login Cloudflare

Buka browser, masuk ke:

https://dash.cloudflare.com

Login atau sign up (gratis)

STEP 1.2: Add Site

1. Klik tombol biru ADD A SITE (kanan atas)

2. Ketik domain lu, contoh:

alltesting.online

3. Klik ADD SITE

4. Pilih plan FREE → klik CONTINUE

STEP 1.3: Copy Nameservers

Cloudflare kasih 2 nameserver, contoh:

elijah.ns.cloudflare.com pola.ns.cloudflare.com

COPY kedua nameserver ini (lu butuh di step berikutnya)

STEP 1.4: Ganti Nameserver di Registrar

Login ke tempat lu beli domain:

🔹 NAMECHEAP:

  1. Domain List → klik MANAGE
  2. Bagian NAMESERVERS → pilih CUSTOM DNS
  3. Paste 2 nameserver Cloudflare
  4. Klik SAVE

🔹 GODADDY:

  1. My Products → Domains → klik DNS
  2. Bagian NAMESERVERS → klik CHANGE
  3. Pilih CUSTOM
  4. Paste 2 nameserver Cloudflare
  5. Klik SAVE

🔹 HOSTINGER:

  1. Domains → klik MANAGE
  2. Bagian NAMESERVERS → klik CHANGE NAMESERVERS
  3. Pilih CUSTOM
  4. Paste 2 nameserver Cloudflare
  5. Klik SAVE
STEP 1.5: Tunggu Propagasi

1. Balik ke Cloudflare → klik DONE, CHECK NAMESERVERS

2. Tunggu 5-60 menit (biasanya 10-15 menit)

3. Domain status jadi ACTIVE (badge hijau)

Cek propagasi:

https://www.whatsmydns.net

Pilih type: NS → ketik domain lu → harus muncul nameserver Cloudflare (hijau semua)
