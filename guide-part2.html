📧 Part 2: Email Routing Setup

⚠️ PENTING:

Part ini MANUAL (Cloudflare requirement). Tapi cuma 1x per email destination.

Setelah verified, semua domain bisa pake email yang sama (unlimited).

2 Setup Email Routing

STEP 2.1: Masuk ke Email Routing
  1. Dashboard Cloudflare → KLIK DOMAIN LU
  2. SIDEBAR KIRI → scroll ke bawah
  3. Cari menu EMAIL (icon amplop ✉️)
  4. Klik EMAIL ROUTING
STEP 2.2: Enable Email Routing

Kalo pertama kali:

  1. Ada tombol GET STARTED atau ENABLE EMAIL ROUTING
  2. Klik tombol itu
  3. Cloudflare auto-create DNS records
  4. Klik ADD RECORDS AND ENABLE
  5. Status jadi ENABLED ✅
STEP 2.3: Add Destination Address
  1. Klik tab DESTINATION ADDRESSES (tab ke-2 dari kiri)
  2. Klik tombol CREATE ADDRESS (tombol biru, kanan atas)
  3. Form muncul
  4. EMAIL ADDRESS: ketik email lu, contoh:
    your-email@gmail.com
  5. Klik SAVE AND SEND VERIFICATION

💡 Email ini bakal jadi inbox tujuan semua forwarded email dari domain-domain lu

STEP 2.4: Verify Email
  1. BUKA INBOX email yang lu masukin tadi
  2. Cari email dari CLOUDFLARE
    Sender: no-reply@notify.cloudflare.com
    Subject: "Verify your email address for Cloudflare Email Routing"
  3. KLIK tombol biru VERIFY EMAIL ADDRESS
  4. Browser redirect ke Cloudflare
  5. Status jadi VERIFIED ✅

⚠️ Ga ada email? Check SPAM/JUNK folder. Atau wait 5-10 menit.

STEP 2.5: Confirm Verified

Balik ke Cloudflare Email Routing dashboard

Tab DESTINATION ADDRESSES → harus muncul:

  • Email: email-lu@gmail.com
  • Status: VERIFIED (badge hijau) ✅

3 Get API Key

STEP 3.1: Go to API Tokens
  1. Klik PROFILE ICON (kanan atas)
  2. Klik MY PROFILE
  3. Sidebar kiri → klik API TOKENS

Atau langsung:

https://dash.cloudflare.com/profile/api-tokens
STEP 3.2: Get Global API Key
  1. Scroll ke bawah
  2. Cari section GLOBAL API KEY
  3. Klik VIEW
  4. Masukin password Cloudflare (confirm identity)
  5. COPY API key-nya (starts with cfk_...)
  6. Save di Notepad/Notes (lu butuh ini buat tool)

✅ SETUP MANUAL SELESAI!

Lu udah punya:

  • ✅ Domain Active di Cloudflare
  • ✅ Destination email verified
  • ✅ Global API Key
