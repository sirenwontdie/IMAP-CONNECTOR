🔑 Gmail App Password

Cara dapetin 16-character App Password buat IMAP auto-verify

⚠️ PENTING: Google sekarang default-nya nyuruh lu pake Passkey (save to device). Lu JANGAN ikutin prompt passkey itu. App Password masih ada, tapi URL-nya disembunyiin. Harus direct-access.
📋 Prerequisite: 2-Step Verification (2FA) harus ON sebelum bisa bikin App Password. Kalau belum on, halaman App Password bakal kosong / blocked.
1 Aktifin 2-Step Verification (kalau belum)
Cek apakah 2FA udah on:
https://myaccount.google.com/signinoptions/twosv
  • Login ke Google account lu
  • Kalau statusnya "ON" → skip ke Step 2
  • Kalau "OFF" → klik "Turn on" → ikutin setup (OTP via SMS/Authenticator)
2 Buka halaman App Password (DIRECT URL)
Ini kunci utamanya. Jangan cari dari menu Google (bakal dilarikan ke Passkey). Buka langsung URL ini di browser:
https://myaccount.google.com/apppasswords
  • Paste URL di atas ke address bar
  • Enter → lu bakal lihat halaman "App passwords"
💡 Kalau redirect ke passkey: Berarti ada passkey aktif di akun lu. Hapus dulu: Security → "How you sign in to Google" → Passkeys → Remove. Setelah dapet App Password, lu bisa enable passkey lagi.
3 Buat App Password baru
Di halaman App Passwords:
  • Ada input "App name" (atau dropdown "Select app")
  • Isi nama apa aja, misal: pinata-imap
  • Klik tombol "Create" (atau "Generate")
Google bakal generate password 16 karakter.
4 Copy 16-character code
Muncul popup/kotak kuning yang nampilin password:
abcd-efgh-ijkl-mnop
  • Copy password ini (format: 4-4-4-4, dipisah dash)
  • Simpan temporer — lu cuma butuh 1x buat paste ke tool
  • Setelah dipake, ga perlu diingat lagi (IMAP creds cuma dipake sekali buat verifikasi)
📌 Catatan: Password ini hanya sekali tampil. Kalau lu lupa copy, harus delete + bikin baru. Tapi ga masalah — bikin App Password unlimited.
5 Paste ke IMAP Connector tool
Balik ke tool:
http://43.134.72.40:4444
  • IMAP Provider: Gmail
  • IMAP Username: email Gmail lu (contoh: remm@gmail.com)
  • IMAP Password: paste 16-char App Password tadi (bukan password Gmail biasa)
Klik "Full Auto Setup" → tool connect ke Gmail IMAP → auto-verify destination.
✅ Setelah selesai: App Password ga perlu lu simpan / ingat. Tool cuma pake 1x buat verifikasi Cloudflare. Setelah itu, Cloudflare yang handle forwarding. Gmail App Password lu bisa lu revoke/delete kalo mau bersihin security.
🔒 Security tips:
• App Password = akses IMAP ke Gmail lu. Jangan share ke siapa2.
• Tool gw ga nyimpan IMAP creds setelah setup selesai (in-memory only, hilang pas restart).
• Setelah setup done, revoke App Password di Google kalo mau extra aman.
• Cloudflare email routing forward email → Gmail lu. Gmail password ga di-share ke Cloudflare.
